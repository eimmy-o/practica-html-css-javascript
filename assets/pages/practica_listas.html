Practica Listas

Practica de Listas en HTML

Listas desordenadas - ul

La etiqueta <ul> se usa para crear listas que no tiene un orden numerico (con viñetas)

Ejemplo:

<ul>
 <li>Elemento</li>
 <li>Elemento</li>
 <li>Elemento</li>
</ul>

Se muestra asi:

  • Elemento
  • Elemento
  • Elemento

Listas ordenadas - ol

La etiqueta <ol> se usa para crear listas numeradas

Ejemplo:

<ol>
 <li>Elemento</li>
 <li>Elemento</li>
 <li>Elemento</li>
</ol>

Se muestra asi:

  1. Elemento
  2. Elemento
  3. Elemento
