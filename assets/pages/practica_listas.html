Practica de Listas en HTML

Listas desordenadas - ul

La etiqueta <ul> se usa para crear listas que no tiene un orden numerico (con viñetas)

Ejemplo:

<ul>
 <li>Elemento</li>
 <li>Elemento</li>
 <li>Elemento</li>
</ul>

Se muestra asi:

  • Elemento
  • Elemento
  • Elemento

Listas ordenadas - ol

La etiqueta <ol> se usa para crear listas numeradas

Ejemplo:

<ol>
 <li>Elemento</li>
 <li>Elemento</li>
 <li>Elemento</li>
</ol>

Se muestra asi:

  1. Elemento
  2. Elemento
  3. Elemento

Listas Anidadas

Las listas anidadas son listas que contienen otras listas dentro de sus elementos. Pueden ser tanto <ul> (no ordenadas) como <ol> (ordenadas). Son útiles para mostrar jerarquías o subcategorías.

Ejemplo:

<ol>
  <li>Frutas</li>
  <ul>
    <li>Manzana</li>
    <li>Banana</li>
    <li>Naranja</li>
  </ul>
  <li>Verduras</li>
  <ul>
    <li>Zanahoria</li>
    <li>Lechuga</li>
  </ul>
</ol>

Se muestra así:

  1. Frutas
    • Manzana
    • Banana
    • Naranja
  2. Verduras
    • Zanahoria
    • Lechuga

Lista de definiciones - dl

Se usa para mostrar términos y sus descripciones.

<dl>
 <dt>HTML</dt>
 <dd>Lenguaje de marcado para crear páginas web.</dd>
 <dt>CSS</dt>
 <dd>Lenguaje para dar estilo al contenido HTML.</dd>
</dl>

Se muestra así:

HTML
Lenguaje de marcado para crear páginas web.
CSS
Lenguaje para dar estilo al contenido HTML.

Atributos útiles

type - ol

Qué hace: Define el tipo de numeración para las listas ordenadas.

Cómo se usa: Dentro de la etiqueta <ol> se agrega el atributo type con valores como "1", "A", "a", "I" o "i".

Ejemplo:

<ol type="A">
  <li>Primero</li>
  <li>Segundo</li>
</ol>

Se muestra así:

  1. Primero
  2. Segundo

list-style-type - ul

Qué hace: Permite definir el tipo de viñeta o numeración mediante CSS.

Cómo se usa: Dentro de una regla de estilo CSS o en línea.

Ejemplo:

<ul style="list-style-type: square;">
  <li>Rojo</li>
  <li>Azul</li>
</ul>

Se muestra así:

  • Rojo
  • Azul

start

Qué hace: Permite definir el número de inicio en una lista ordenada.

Cómo se usa: Agregando el atributo start en la etiqueta <ol>.

Ejemplo:

<ol start="5">
  <li>Paso cinco</li>
  <li>Paso seis</li>
</ol>

Se muestra así:

  1. Paso cinco
  2. Paso seis

reversed

Qué hace: Invierte el orden de numeración en una lista ordenada.

Cómo se usa: Añadiendo el atributo reversed sin valor dentro de <ol>.

Ejemplo:

<ol reversed>
  <li>Uno</li>
  <li>Dos</li>
</ol>

Se muestra así:

  1. Uno
  2. Dos
