Práctica de CSS

CSS nos ayuda o sirve para dar estilo y diseño a nuestra página web.
Para aplicarlo tenemos tres opciones:

1️En línea (código embebido dentro de las etiquetas HTML).

2️Interno (código dentro del <style> en la cabecera).

3️Externo (archivo .css enlazado con <link>).

Modelo de Caja

Modelo caja #1:
Todo elemento HTML se presenta en pantalla como una caja que tiene contenido, padding, borde y margen.

Modelo caja #2:
Las cajas se pueden modificar con propiedades CSS como margin, padding, border y width.

Notas importantes:

1️CSS se ejecuta de arriba hacia abajo, las reglas más recientes sobrescriben las anteriores.

2️Tener cuidado con el orden y la jerarquía de selectores.

3️Es recomendable aplicar un “reset” de márgenes y paddings antes de diseñar.
