Practica de Imagenes en HTML

Etiqueta img

La etiqueta <img> se usa para insertar imagenes en una pagina web

Pero no se puede usar por si sola, necesita un atributo esencial: src (ruta o enlace de la imagen que queremos insertar)

Ejemplo:

<img src="imagen.jpg">
